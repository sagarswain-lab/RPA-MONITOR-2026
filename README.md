## 🌌 Overview

**RPA MONITOR 2026** is a state-of-the-art, high-frequency enterprise telemetry monitoring dashboard engineered for maximum performance, precision, and aesthetic brilliance. Built specifically to handle continuous real-time data streams processing **50,000+ automation telemetry records** at **200ms flush cycles**, this application delivers zero-latency interactive grid manipulation completely on the client side without relying on external UI grid frameworks.

Designed with a **Cyberpunk Cinematic Dark UI System**, featuring glowing neon indicators, real-time pulse alerts, modular view toggles, zero-lag virtualization, and instantaneous multi-column data queries.

---

## ⚡ Key Features & Architecture

### 🚀 Custom Fixed-Node Virtualization Engine (60 FPS Locked)
- **Zero External Grid Libraries**: Built from scratch using raw DOM manipulation and RAF sync.
- **Fixed DOM Footprint**: Recycles a static pool of ~25 DOM nodes regardless of dataset volume (10 rows or 50,000+ rows).
- **Sub-frame Translation**: Uses `transform: translateY()` positioning with zero layout recalculation overhead.
- **Overscan Buffering**: Smooth, tear-free scrolling experience.

### 📊 Dynamic Data Pipeline & Real-Time Engine
- **Telemetry Processing**: Ingests high-frequency CSV data stream batches every 200ms via `StateEngine`.
- **Live KPI Accumulators**: Bypasses React state overhead by imperatively updating KPI counters directly via DOM references.
- **Alert Detection System**: Real-time visual pulse animations (`row--alert`) triggered by anomalous conditions (e.g. ROI anomalies, failed bot executions).

### 🎛️ Advanced Query & Data Manipulation
- **Multi-Column Sorting (Feature 4 + 9)**: 
  - Single Click: Primary column sort with direction toggles (Ascending/Descending).
  - Shift + Click: Stacks secondary and tertiary multi-column sorting priorities with active rank badges.
- **Categorical Multi-Select Filters (Feature 7)**: Interactive floating dropdowns for filtering by Automation Type, Department, and Industry.
- **Fuzzy Search Engine (Feature 10)**: Instantaneous tokenized search across project names, IDs, partners, and countries.
- **Custom Layout Manager (Feature 6)**: Modular UI system allowing operators to toggle KPI Bars, Filters, and Search interfaces on-the-fly.

## 🛠️ Tech Stack & Engineering Standards

- **Core Framework**: React 18 (Hooks, `useRef`, `useCallback`, `useImperativeHandle`, `forwardRef`)
- **Build Tool**: Vite 5.4 (Instant HMR & optimized production bundler)
- **Styling System**: Pure Vanilla CSS3 (Custom CSS Properties, Glassmorphism, Neon Cyberpunk Aesthetics, Flexbox/Grid)
- **Data Engine**: Pure JavaScript Class (`StateEngine.js`) with Event Bus Architecture
- **Performance Hardware Acceleration**: CSS `will-change`, GPU-accelerated transforms, and `requestAnimationFrame` locking

## 🚀 Getting Started (in windows terminal)

### Prerequisites
- Node.js (v18.0.0 or higher recommended)
- npm or yarn

### Installation & Execution

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sagarswain-lab/RPA-MONITOR-2026.git
   cd RPA-MONITOR-2026
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the development server**:
   ```bash
   npm run dev
   ```

4. **Open in Browser**:
   Navigate to `https://rpa-monitor-2026.vercel.app/` (or local `http://localhost:5173/`).
